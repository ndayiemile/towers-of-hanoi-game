
# Towers of Hanoi Game

This project implements the classic Towers of Hanoi puzzle in Java.

## How to Run

Compile and run the program, providing the number of rings as an argument:

```
javac Towers.java
java Towers <number_of_rings>
```

Example for 3 rings:

```
java Towers 3
```

## Files
- `Towers.java`: Main implementation
